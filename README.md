## Проект

Сапёр с графическим интерфейсом на PyQt.

Игрок открывает ячейки поля, помечает мины флагами, выигрывает, если открыты все безопасные клетки.  

## Библиотеки

- PyQt6

## Фичи по итерациям

### Итерация 1
- Создание главного окна, панели инструментов, игрового поля
- Обработка событий мыши (левый клик – открыть, правый – поставить флаг)

### Итерация 2
- Таймер на QTimer с отсчётом времени игры
- Хранение и отображение рекордов (топ-20 для каждой сложности)
- Кастомные диалоговые окна для победы/поражения
- Меню настройки сложности и своя сложность
- Темы оформления через QSS (стандартная, тёмная, цветная, загрузка из файла)
- Юнит-тесты (pytest) и профилирование основных функций

## Структура проекта

| Файл | Назначение |
|------|------------|
| `config.py` | Константы, dataclass `GameConfig`, исключения |
| `records.py` | Модуль хранения рекордов (JSON) |
| `feature1_ui.py` | Базовый интерфейс: окно, меню, тулбар, сетка |
| `feature2_game.py` | Игровая логика: мины, флаги, открытие клеток |
| `feature3_timer.py` | Таймер, запускаемый первым кликом |
| `feature4_dialogs.py` | Диалоги завершения и произвольной сложности |
| `feature5_qss.py` | Меню «Вид» для выбора тем QSS |
| `feature6_records.py` | Интеграция рекордов в интерфейс (фича ассистента) |
| `main.py` | Точка входа в приложение |
| `tests/` | Тесты для `records` и игровой логики |
| `python profile_game.py` | Профилирование производительности |
